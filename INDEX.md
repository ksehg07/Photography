## ✨ Key Features

**Design System**
- 4 CSS variables for cinematic color palette
- Minimalist, visual-first aesthetic
- Fully customizable

**Animations**
- FadeInView component for viewport-triggered animations
- Mobile marquee with CSS @keyframes
- Performance optimized

**Responsive**
- Mobile-first design
- Desktop grid gallery
- Mobile/tablet infinite scroll marquee
- Breakpoints: 480px, 768px

**Technology**
- Next.js 14 with App Router
- TypeScript with strict mode
- CSS Modules (no utilities)
- Framer Motion
- Next.js Image optimization
